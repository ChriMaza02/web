Arquitectura del Computador

Partes Básicas del Computador

Unidad Central de Procesamiento (CPU)

La CPU, también conocida como el cerebro del computador, es responsable de ejecutar instrucciones y procesar datos. Se encarga de realizar cálculos aritméticos, lógicos y de control.

Tipos de CPU:

  • CPU de un solo núcleo: Procesa una tarea a la vez.
  • CPU multinúcleo: Contiene múltiples núcleos, lo que permite realizar varias tareas simultáneamente. Ejemplos incluyen dual-core, quad-core, hexa-core, y octa-core.
  • CPU de 32 bits: Procesa datos en fragmentos de 32 bits.
  • CPU de 64 bits: Procesa datos en fragmentos de 64 bits, permitiendo manejar más memoria y ejecutar aplicaciones más avanzadas.

Memoria (RAM)

La memoria RAM (Memoria de Acceso Aleatorio) es un tipo de almacenamiento volátil que almacena temporalmente los datos y programas que el CPU necesita mientras el computador está encendido. Permite el acceso rápido a los datos.

Tipos de RAM:

  • DDR (Double Data Rate): DDR, DDR2, DDR3, DDR4, y DDR5 son versiones de RAM, cada una con mayor velocidad y eficiencia que la anterior.
  • SRAM (Static RAM): Más rápida y cara que la DRAM, usada en cachés de CPU.
  • DRAM (Dynamic RAM): Más común y más lenta que la SRAM, usada en la mayoría de las memorias principales.

Disco Duro (HDD/SSD)

El disco duro es un dispositivo de almacenamiento permanente donde se guardan los datos y programas. Los discos duros pueden ser de tipo HDD (unidad de disco duro) o SSD (unidad de estado sólido), siendo estos últimos más rápidos y resistentes.

Tipos de discos duros:

  • HDD (Hard Disk Drive): Utiliza platos giratorios y cabezales magnéticos para leer y escribir datos. Más económico pero más lento y frágil.
  • SSD (Solid State Drive): Utiliza memoria flash para almacenar datos, sin partes móviles. Es más rápido, duradero y consume menos energía.
  • NVMe (Non-Volatile Memory Express): Un tipo de SSD que utiliza el bus PCIe para ofrecer velocidades de transferencia aún mayores.

Placa Base (Motherboard)

La placa base es el componente principal que conecta todos los demás componentes del computador. Proporciona las conexiones eléctricas y de datos entre la CPU, la memoria, los dispositivos de almacenamiento y otros periféricos.

Tipos de placas base:

  • ATX (Advanced Technology eXtended): El formato más común, ofrece más espacio para componentes y expansiones.
  • Micro-ATX: Más pequeña que la ATX, con menos opciones de expansión pero más compacta.
  • Mini-ITX: Aún más pequeña, ideal para sistemas compactos, pero con limitaciones en expansión y rendimiento térmico.

Unidad de Procesamiento Gráfico (GPU)

La GPU es responsable de procesar y renderizar imágenes, videos y gráficos 3D. Es esencial para tareas que requieren procesamiento intensivo de gráficos, como juegos, diseño gráfico y edición de video.

Tipos de GPU:

  • Integrada: Integrada en la CPU o la placa base, adecuada para tareas básicas y multimedia.
  • Dedicada: Una tarjeta gráfica separada con su propia memoria VRAM, utilizada para juegos y aplicaciones gráficas avanzadas.
  • GPU para estaciones de trabajo: Diseñadas para aplicaciones profesionales de diseño y cálculo intensivo, como la renderización 3D y la edición de video.

Fuente de Alimentación

La fuente de alimentación convierte la corriente alterna (CA) de la red eléctrica en corriente continua (CC) que alimenta todos los componentes del computador. Asegura que cada componente reciba el voltaje adecuado.

Tipos de fuentes de alimentación:

  • ATX: El estándar más común para computadoras de escritorio, ofrece varios conectores y capacidades de potencia.
  • SFX (Small Form Factor): Más compacta, utilizada en sistemas pequeños y mini PCs.
  • Modular: Permite desconectar cables innecesarios para mejorar la gestión del cableado y el flujo de aire.

Periféricos

Los periféricos son dispositivos externos que se conectan al computador, como el teclado, el ratón, la impresora y el monitor. Permiten la interacción con el sistema y la entrada/salida de datos.

Tipos de periféricos:

  • Entrada: Dispositivos como el teclado, ratón, escáner y micrófono que permiten introducir datos al computador.
  • Salida: Dispositivos como el monitor, impresora, altavoces y auriculares que permiten recibir información del computador.
  • Entrada/Salida: Dispositivos como las unidades USB, discos duros externos y pantallas táctiles que permiten tanto la entrada como la salida de datos.
